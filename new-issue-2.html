Issue Name: Not valid encoding

Issue Type: Error

Issue Description:

Trying to validate a simple html file and write things in head like this:
<head>
<title> test </title>
</head>
 But get error message of: The character encoding was not declared. Proceeding using windows-1252.


Issue level:

This issue has been resolved.

Solution: Input the line "<meta charset="UTF-8">" to give a valid encoding of the file. Or the validator can't get it.
